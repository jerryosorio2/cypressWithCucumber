# Cypress With Cucumber

Cypress With Cucumber Examples

## Installation

First clone repository

```bash
git clone https://github.com/jerryosorio2/cypressWithCucumber.git
```

Install packages

```
npm install
```

## Execution

Execute via test script

```bash
npm run <test script>
```
